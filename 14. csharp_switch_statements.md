# C# Switch Statements

The `switch` statement is used to select one of many code blocks to execute based on the value of an expression.

---

# Syntax

```csharp
switch(expression) 
{
    case x:
        // code block
        break;

    case y:
        // code block
        break;

    default:
        // code block
        break;
}
```

---

# How switch Works

1. The `expression` is evaluated once.
2. The value is compared with each `case`.
3. If a match is found, the corresponding code block runs.
4. The `break` statement stops further checking.
5. The `default` block runs if no match is found.

---

# Example: Weekday Name

```csharp
int day = 4;

switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;

    case 2:
        Console.WriteLine("Tuesday");
        break;

    case 3:
        Console.WriteLine("Wednesday");
        break;

    case 4:
        Console.WriteLine("Thursday");
        break;

    case 5:
        Console.WriteLine("Friday");
        break;

    case 6:
        Console.WriteLine("Saturday");
        break;

    case 7:
        Console.WriteLine("Sunday");
        break;
}
```

### Output

```text
Thursday
```

---

# The `break` Keyword

The `break` keyword exits the `switch` block.

Without `break`, execution may continue into the next case.

## Example

```csharp
int number = 2;

switch (number)
{
    case 1:
        Console.WriteLine("One");
        break;

    case 2:
        Console.WriteLine("Two");
        break;

    case 3:
        Console.WriteLine("Three");
        break;
}
```

### Output

```text
Two
```

---

# The `default` Keyword

The `default` block executes when no case matches the expression.

## Example

```csharp
int day = 4;

switch (day)
{
    case 6:
        Console.WriteLine("Today is Saturday.");
        break;

    case 7:
        Console.WriteLine("Today is Sunday.");
        break;

    default:
        Console.WriteLine("Looking forward to the Weekend.");
        break;
}
```

### Output

```text
Looking forward to the Weekend.
```

---

# Multiple Cases

You can combine multiple cases to execute the same code.

## Example

```csharp
char grade = 'B';

switch (grade)
{
    case 'A':
    case 'B':
        Console.WriteLine("Good Job");
        break;

    case 'C':
        Console.WriteLine("Average");
        break;

    default:
        Console.WriteLine("Needs Improvement");
        break;
}
```

### Output

```text
Good Job
```

# Real-World Example

## Menu Selection

```csharp
int choice = 2;

switch (choice)
{
    case 1:
        Console.WriteLine("Add New Student");
        break;

    case 2:
        Console.WriteLine("View Students");
        break;

    case 3:
        Console.WriteLine("Delete Student");
        break;

    default:
        Console.WriteLine("Invalid Choice");
        break;
}
```

---

# switch vs if-else

| switch | if-else |
|---|---|
| Better for many fixed values | Better for complex conditions |
| Cleaner and easier to read | More flexible |
| Faster in some cases | Good for ranges and logical expressions |

## Exercise 2

Create a calculator using `switch`.

### Example

```csharp
char op = '+';
int a = 10;
int b = 5;

switch (op)
{
    case '+':
        Console.WriteLine(a + b);
        break;

    case '-':
        Console.WriteLine(a - b);
        break;

    case '*':
        Console.WriteLine(a * b);
        break;

    case '/':
        Console.WriteLine(a / b);
        break;

    default:
        Console.WriteLine("Invalid Operator");
        break;
}
```

### Output

```text
15
```
