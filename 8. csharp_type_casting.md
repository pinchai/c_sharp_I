# C# Type Casting

Type casting is the process of converting one data type into another data type.

In C#, there are **two types of casting**:

1. **Implicit Casting** (Automatic)
2. **Explicit Casting** (Manual)

---

# 1. Implicit Casting (Automatic)

Implicit casting happens automatically when converting a **smaller type** into a **larger type**.

## Conversion Order

```csharp
char -> int -> long -> float -> double
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int myInt = 9;
        double myDouble = myInt;

        Console.WriteLine(myInt);      // 9
        Console.WriteLine(myDouble);   // 9
    }
}
```

## Explanation

- `int` is smaller than `double`
- C# automatically converts `int` to `double`
- No data loss occurs

---

# 2. Explicit Casting (Manual)

Explicit casting is used when converting a **larger type** into a **smaller type**.

## Conversion Order

```csharp
double -> float -> long -> int -> char
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        double myDouble = 9.78;
        int myInt = (int)myDouble;

        Console.WriteLine(myDouble);   // 9.78
        Console.WriteLine(myInt);      // 9
    }
}
```

## Explanation

- `double` contains decimal values
- `int` only stores whole numbers
- Decimal part `.78` is removed

---

# Data Loss During Casting

Explicit casting may cause data loss.

## Example

```csharp
double price = 99.99;
int newPrice = (int)price;

Console.WriteLine(newPrice); // 99
```

The decimal value `.99` is lost.

# Type Conversion Methods

C# provides built-in conversion methods using the `Convert` class.

---

## Convert.ToString()

Convert another type into a string.

### Example

```csharp
int number = 100;

string result = Convert.ToString(number);

Console.WriteLine(result);
Console.WriteLine(result.GetType());
```

---

## Convert.ToDouble()

Convert another type into double.

### Example

```csharp
int number = 50;

double result = Convert.ToDouble(number);

Console.WriteLine(result);
```

---

## Convert.ToInt32()

Convert another type into int.

### Example

```csharp
double score = 88.75;

int result = Convert.ToInt32(score);

Console.WriteLine(result);
```

### Output

```text
89
```

> `Convert.ToInt32()` rounds the number.

---

## Convert.ToBoolean()

Convert value into boolean.

### Example

```csharp
string value = "true";

bool result = Convert.ToBoolean(value);

Console.WriteLine(result);
```

---

## Convert.ToInt64()

Convert value into long.

### Example

```csharp
int number = 100;

long result = Convert.ToInt64(number);

Console.WriteLine(result);
```

---

# Difference Between Casting and Convert

| Feature | Casting | Convert |
|---|---|---|
| Syntax | `(int)value` | `Convert.ToInt32(value)` |
| Decimal Handling | Removes decimal | Rounds decimal |
| Null Handling | Error possible | Safer |
| Performance | Faster | Slightly slower |

---

# Example: Casting vs Convert

```csharp
using System;

class Program
{
    static void Main()
    {
        double number = 9.78;

        int castValue = (int)number;
        int convertValue = Convert.ToInt32(number);

        Console.WriteLine(castValue);     // 9
        Console.WriteLine(convertValue);  // 10
    }
}
```

# Real-World Example

User input from `Console.ReadLine()` is always a string.

You must convert it into a number before calculation.

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("Enter age: ");
        string ageText = Console.ReadLine();

        int age = Convert.ToInt32(ageText);

        Console.WriteLine("Your age is: " + age);
    }
}
```

# Common Errors

## Invalid Conversion

```csharp
string text = "Hello";

int number = Convert.ToInt32(text);
```

## Result

```text
FormatException
```

Because `"Hello"` is not a number.

---

# Safe Conversion with TryParse

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("Enter number: ");

        string input = Console.ReadLine();

        bool status = int.TryParse(input, out int result);

        if (status)
        {
            Console.WriteLine("Valid Number");
            Console.WriteLine(result);
        }
        else
        {
            Console.WriteLine("Invalid Number");
        }
    }
}
```
