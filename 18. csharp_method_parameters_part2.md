# C# Method Parameters

Methods in C# can accept values called **parameters**.  
When calling a method, the values you pass are called **arguments**.

---

# 1. Parameters and Arguments

- **Parameter** → variable inside the method definition
- **Argument** → actual value passed to the method

## Syntax

```csharp
methodName(argument);
```

## Example

```csharp
using System;

class Program
{
    static void MyMethod(string name) // parameter
    {
        Console.WriteLine("Hello " + name);
    }

    static void Main()
    {
        MyMethod("John"); // argument
        MyMethod("Sara");
    }
}
```

## Output

```text
Hello John
Hello Sara
```

---

# 2. Multiple Parameters

You can pass multiple parameters into a method.

## Example

```csharp
using System;

class Program
{
    static void StudentInfo(string name, int age)
    {
        Console.WriteLine(name + " is " + age + " years old.");
    }

    static void Main()
    {
        StudentInfo("David", 20);
        StudentInfo("Anna", 22);
    }
}
```

## Output

```text
David is 20 years old.
Anna is 22 years old.
```

---

# 3. Default Parameter Value

You can assign a default value to a parameter.  
If no argument is passed, the default value will be used.

## Example

```csharp
using System;

class Program
{
    static void Country(string country = "Cambodia")
    {
        Console.WriteLine(country);
    }

    static void Main()
    {
        Country("Japan");
        Country("France");
        Country();
    }
}
```

## Output

```text
Japan
France
Cambodia
```

---

# 4. Return Values

Methods can return values using the `return` keyword.

## Example

```csharp
using System;

class Program
{
    static int AddNumbers(int x, int y)
    {
        return x + y;
    }

    static void Main()
    {
        int result = AddNumbers(5, 3);

        Console.WriteLine(result);
    }
}
```

## Output

```text
8
```

---

# 5. Named Arguments

Named arguments allow you to specify parameter names when calling a method.

This improves readability and allows arguments in different order.

## Example

```csharp
using System;

class Program
{
    static void PersonInfo(string name, int age, string country)
    {
        Console.WriteLine(name + " - " + age + " - " + country);
    }

    static void Main()
    {
        PersonInfo(age: 25, country: "Canada", name: "Alex");
    }
}
```

## Output

```text
Alex - 25 - Canada
```

# Summary Table

| Feature | Description |
|---|---|
| Parameters | Variables inside method definition |
| Arguments | Values passed to methods |
| Multiple Parameters | Pass more than one value |
| Default Parameter | Uses default value if no argument given |
| Return Value | Method sends value back using `return` |
| Named Arguments | Specify parameter names during method call |
