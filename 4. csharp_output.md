# C# Output

To display output or print text in C#, you can use the `WriteLine()` method from the `Console` class.

## Using `Console.WriteLine()`

The `WriteLine()` method prints text and then moves the cursor to the next line.

### Example

```csharp
Console.WriteLine("Hello World!");
```

### Output

```text
Hello World!
```

---

## Multiple `WriteLine()` Statements

You can use multiple `WriteLine()` methods.  
Each statement prints on a new line.

### Example

```csharp
Console.WriteLine("Hello World!");
Console.WriteLine("I am Learning C#");
Console.WriteLine("It is awesome!");
```

### Output

```text
Hello World!
I am Learning C#
It is awesome!
```

---

## Printing Numbers and Calculations

`WriteLine()` can also display numbers and mathematical expressions.

### Example

```csharp
Console.WriteLine(3 + 3);
```

### Output

```text
6
```

---

# The `Write()` Method

C# also provides the `Write()` method.

The difference between `Write()` and `WriteLine()` is:

| Method | New Line |
|---|---|
| `WriteLine()` | Yes |
| `Write()` | No |

---

## Example of `Write()`

```csharp
Console.Write("Hello World! ");
Console.Write("I will print on the same line.");
```

### Output

```text
Hello World! I will print on the same line.
```

> Notice the space after `"Hello World!"` to improve readability.

---

# Summary

| Method | Description |
|---|---|
| `Console.WriteLine()` | Prints output and moves to a new line |
| `Console.Write()` | Prints output on the same line |

In most beginner examples, `WriteLine()` is commonly used because the output is easier to read.
