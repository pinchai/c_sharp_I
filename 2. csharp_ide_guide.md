# C# IDE

The easiest way to get started with C# is to use an IDE.

An IDE (**Integrated Development Environment**) is used to write, edit, compile, and run code.

A popular IDE for C# is Visual Studio Community, which is free to download from:

https://visualstudio.microsoft.com/vs/community/

Applications written in C# use the .NET platform, so using Visual Studio is a good choice because both C# and .NET are developed by Microsoft.

---

# Installing C#

## Step 1: Download Visual Studio

Download and install Visual Studio Community Edition from the official website.

## Step 2: Choose the .NET Workload

After opening the Visual Studio Installer:

- Select the **.NET desktop development** workload
- Click **Install** or **Modify**

This installs the tools required for C# development.

---

# Creating Your First C# Project

## Step 1: Launch Visual Studio

After installation, click **Launch**.

## Step 2: Create a New Project

On the start screen:

- Click **Create a new project**

## Step 3: Select Project Type

Choose:

- **Console App (.NET)**

Then click **Next**.

## Step 4: Name Your Project

Enter a project name such as:

```text
HelloWorld
```

Then click **Create**.

---

# Your First C# Program

Visual Studio automatically creates a file named:

```text
Program.cs
```

Example code:

```csharp
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }
}
```

---

# Understanding the Code

## `using System;`

Imports built-in classes and functions from the System namespace.

## `namespace HelloWorld`

Groups related code together.

## `class Program`

Defines a class named `Program`.

## `static void Main(string[] args)`

The main entry point of the application.

## `Console.WriteLine()`

Displays text on the console screen.

Example:

```csharp
Console.WriteLine("Hello World!");
```

Output:

```text
Hello World!
```

---

# Running the Program

To run the program:

- Press **F5**
- Or click:

```text
Debug → Start Debugging
```

Visual Studio will:

1. Compile the code
2. Execute the application
3. Display the result in the console

Example output:

```text
Hello World!
```
