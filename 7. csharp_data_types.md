# C# Data Types

In C#, every variable must have a **data type**.  
A data type defines:

- What kind of value the variable can store
- How much memory it uses
- What operations can be performed on it

---

## Basic Example

```csharp
int myNum = 5;               // Integer (whole number)
double myDoubleNum = 5.99D;  // Floating point number
char myLetter = 'D';         // Character
bool myBool = true;          // Boolean
string myText = "Hello";     // String
```

---

# Common C# Data Types

| Data Type | Size | Description |
|---|---|---|
| `int` | 4 bytes | Stores whole numbers |
| `long` | 8 bytes | Stores very large whole numbers |
| `float` | 4 bytes | Stores decimal numbers (6–7 digits precision) |
| `double` | 8 bytes | Stores decimal numbers (15 digits precision) |
| `bool` | 1 byte | Stores `true` or `false` |
| `char` | 2 bytes | Stores a single character |
| `string` | 2 bytes per character | Stores text |

---

# Number Types

Number types are divided into two groups:

## 1. Integer Types
Store whole numbers without decimals.

Examples:

```text
10
-5
2000
```

Types:
- `int`
- `long`

---

## 2. Floating Point Types
Store decimal numbers.

Examples:

```text
3.14
99.99
-10.5
```

Types:
- `float`
- `double`

---

# Integer Types

## int

Used for normal whole numbers.

Range:

```text
-2,147,483,648 to 2,147,483,647
```

Example:

```csharp
int myNum = 100000;
Console.WriteLine(myNum);
```

Output:

```text
100000
```

---

## long

Used for very large whole numbers.

You must add `L` at the end of the value.

Example:

```csharp
long myNum = 15000000000L;
Console.WriteLine(myNum);
```

Output:

```text
15000000000
```

---

# Floating Point Types

Used for decimal numbers.

---

## float

- Precision: about 6–7 digits
- Must end with `F`

Example:

```csharp
float myNum = 5.75F;
Console.WriteLine(myNum);
```

Output:

```text
5.75
```

---

## double

- Precision: about 15 digits
- Must end with `D` (optional but recommended)

Example:

```csharp
double myNum = 19.99D;
Console.WriteLine(myNum);
```

Output:

```text
19.99
```

---

# float vs double

| Feature | float | double |
|---|---|---|
| Size | 4 bytes | 8 bytes |
| Precision | 6–7 digits | 15 digits |
| Suffix | `F` | `D` |
| Recommended | Small calculations | Most calculations |

Example:

```csharp
float price1 = 9.1234567F;
double price2 = 9.12345678912345D;

Console.WriteLine(price1);
Console.WriteLine(price2);
```

---

# Scientific Numbers

C# supports scientific notation using `e` or `E`.

Example:

```csharp
float f1 = 35e3F;
double d1 = 12E4D;

Console.WriteLine(f1);
Console.WriteLine(d1);
```

Output:

```text
35000
120000
```

---

# Boolean Type

The `bool` type stores only:

- `true`
- `false`

Example:

```csharp
bool isCSharpFun = true;
bool isFishTasty = false;

Console.WriteLine(isCSharpFun);
Console.WriteLine(isFishTasty);
```

Output:

```text
True
False
```

---

# Character Type

The `char` type stores a single character.

Rules:
- Use single quotes `' '`
- Only one character allowed

Example:

```csharp
char myGrade = 'B';

Console.WriteLine(myGrade);
```

Output:

```text
B
```

---

# String Type

The `string` type stores text.

Rules:
- Use double quotes `" "`

Example:

```csharp
string greeting = "Hello World";

Console.WriteLine(greeting);
```

Output:

```text
Hello World
```

# Interview Questions

## 1. What is a data type in C#?

A data type defines the type of value a variable can store.

---

## 2. Difference between `float` and `double`?

- `float` has lower precision
- `double` has higher precision and is commonly used

---

## 3. Difference between `char` and `string`?

| char | string |
|---|---|
| Single character | Multiple characters |
| `'A'` | `"ABC"` |

---

## 4. Why use `long` instead of `int`?

Use `long` when the number is too large for `int`.
