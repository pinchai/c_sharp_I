# C# Strings

## String Concatenation

String concatenation means combining strings together.

### Using `+`

```csharp
string firstName = "John";
string lastName = "Doe";

string fullName = firstName + " " + lastName;

Console.WriteLine(fullName);
```

### Using `Concat()`

```csharp
string firstName = "John";
string lastName = "Doe";

string fullName = string.Concat(firstName, " ", lastName);

Console.WriteLine(fullName);
```

---

# String Interpolation

String interpolation uses `$` to insert variables directly into a string.

```csharp
string name = "Alice";
int age = 20;

string result = $"My name is {name} and I am {age} years old.";

Console.WriteLine(result);
```

# Access Strings

Strings are arrays of characters.

## Access Character by Index

```csharp
string text = "Hello";

Console.WriteLine(text[0]);
Console.WriteLine(text[1]);
```

## Loop Through String

```csharp
string text = "Hello";

foreach (char c in text)
{
    Console.WriteLine(c);
}
```

---

# Strings - Special Characters

| Escape Sequence | Description |
|---|---|
| `\"` | Double quote |
| `\'` | Single quote |
| `\\` | Backslash |
| `\n` | New line |
| `\t` | Tab |

## Example

```csharp
Console.WriteLine("Hello \"World\"");
Console.WriteLine("Line1\nLine2");
Console.WriteLine("Name\tAge");
```

# Built-in String Functions

## Length

```csharp
string text = "Hello";

Console.WriteLine(text.Length);
```

## ToUpper()

```csharp
string text = "hello";

Console.WriteLine(text.ToUpper());
```

## ToLower()

```csharp
string text = "HELLO";

Console.WriteLine(text.ToLower());
```

## Trim()

```csharp
string text = "   Hello   ";

Console.WriteLine(text.Trim());
```

## Contains()

```csharp
string text = "Hello World";

Console.WriteLine(text.Contains("World"));
```

## StartsWith()

```csharp
string text = "Hello World";

Console.WriteLine(text.StartsWith("Hello"));
```

## EndsWith()

```csharp
string text = "Hello World";

Console.WriteLine(text.EndsWith("World"));
```

## IndexOf()

```csharp
string text = "Hello World";

Console.WriteLine(text.IndexOf("World"));
```

## Substring()

```csharp
string text = "Hello World";

Console.WriteLine(text.Substring(0, 5));
```

# Real-World Example

```csharp
Console.Write("Enter your first name: ");
string firstName = Console.ReadLine();

Console.Write("Enter your last name: ");
string lastName = Console.ReadLine();

string fullName = $"{firstName} {lastName}";

Console.WriteLine($"Welcome {fullName.ToUpper()}");
```
