# C# Variables

Variables are containers used to store data values in a program.

In C#, different data types are used for different kinds of data.

## Common Variable Types

| Data Type | Description | Example |
|---|---|---|
| `int` | Stores whole numbers | `123`, `-50` |
| `double` | Stores decimal numbers | `19.99`, `-3.5` |
| `char` | Stores a single character | `'A'`, `'b'` |
| `string` | Stores text | `"Hello"` |
| `bool` | Stores `true` or `false` | `true`, `false` |

---

# Declaring (Creating) Variables

To create a variable, specify:

1. The data type  
2. The variable name  
3. The value  

## Syntax

```csharp
type variableName = value;
```

### Example

```csharp
string name = "John";
Console.WriteLine(name);
```

### Output

```text
John
```

---

# Integer Variable Example

```csharp
int myNum = 15;
Console.WriteLine(myNum);
```

### Output

```text
15
```

---

# Declare First, Assign Later

You can declare a variable first and assign the value later.

```csharp
int myNum;
myNum = 15;

Console.WriteLine(myNum);
```

### Output

```text
15
```

---

# Changing Variable Values

You can change the value of a variable after it has been created.

```csharp
int myNum = 15;

myNum = 20;

Console.WriteLine(myNum);
```

### Output

```text
20
```

---

---

# Complete Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int age = 25;
        double salary = 1500.75;
        char grade = 'A';
        bool isStudent = true;
        string name = "Dara";

        Console.WriteLine(name);
        Console.WriteLine(age);
        Console.WriteLine(salary);
        Console.WriteLine(grade);
        Console.WriteLine(isStudent);
    }
}
```

### Output

```text
Dara
25
1500.75
A
True
```

# Important Notes

- Variable names should be meaningful.
- C# is case-sensitive.

Example:

```csharp
int age = 20;
int Age = 30;
```

`age` and `Age` are different variables.

---

# Naming Rules for Variables

✅ Allowed:

```csharp
int myAge = 20;
string first_name = "John";
double price1 = 10.5;
```

❌ Not Allowed:

```csharp
int 1age = 20;
string first-name = "John";
```

---

# Summary

- Variables store data.
- Every variable must have a data type.
- Values can be changed after creation.
- C# supports many data types such as `int`, `double`, `char`, `string`, and `bool`.
