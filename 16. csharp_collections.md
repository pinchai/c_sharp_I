# C# Collections

## 1. Array

-   **Length**: Returns the number of elements
-   **Array.Sort()**: Sorts the array
-   **Array.Reverse()**: Reverses the array
-   **Array.IndexOf()**: Finds the index of an element
-   **Array.Clear()**: Clears values

``` csharp
int[] numbers = {30,10,20};
Console.WriteLine(numbers.Length);
Array.Sort(numbers);
foreach(var n in numbers)
    Console.WriteLine(n);
```

## 2. List`<T>`

-   **Add()**
-   **AddRange()**
-   **Remove()**
-   **RemoveAt()**
-   **Insert()**
-   **Contains()**
-   **Sort()**
-   **Clear()**
-   **Count**

``` csharp
List<string> names = new List<string>();
names.Add("John");
names.Add("Dara");
names.Add("Sok");
Console.WriteLine(names.Count);
names.Remove("Dara");
```

## 3. Dictionary\<TKey,TValue\>

-   **Add()**
-   **Remove()**
-   **ContainsKey()**
-   **ContainsValue()**
-   **TryGetValue()**
-   **Clear()**
-   **Count**

``` csharp
Dictionary<int,string> students = new();
students.Add(1,"John");
students.Add(2,"Dara");
Console.WriteLine(students.ContainsKey(2));
Console.WriteLine(students[2]);
```

## 4. Queue`<T>`

-   **Enqueue()**
-   **Dequeue()**
-   **Peek()**
-   **Contains()**
-   **Clear()**
-   **Count**

``` csharp

q.Enqueue("A");
q.Enqueue("B");
q.Enqueue("C");
Console.WriteLine(q.Dequeue());
Console.WriteLine(q.Peek());
```
