# C# Collections

Collections in C# are used to store, manage, and manipulate groups of data.

They help developers:
- Store multiple values
- Add or remove data dynamically
- Search and sort data
- Process large amounts of information efficiently

---

# Description

## 1. Array

An **Array** stores multiple values of the same type in a fixed-size structure.

### Features
- Fixed size
- Fast access using index
- Same data type only

### Example

```csharp
using System;

class Program
{
    static void Main()
    {
        string[] fruits = { "Apple", "Banana", "Orange" };

        Console.WriteLine(fruits[0]);
    }
}
```

### Output

```text
Apple
```

---

## 2. List<T>

A `List<T>` is a dynamic collection that can grow or shrink.

### Features
- Dynamic size
- Easy to add/remove items
- Generic collection

### Example

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List<string> names = new List<string>();

        names.Add("John");
        names.Add("Dara");
        names.Add("Sok");

        Console.WriteLine(names[1]);
    }
}
```

### Output

```text
Dara
```

---

## 3. Dictionary<TKey, TValue>

Stores data in **key-value pairs**.

### Features
- Fast searching
- Unique keys
- Similar to JSON objects

### Example

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary<int, string> students =
            new Dictionary<int, string>();

        students.Add(1, "John");
        students.Add(2, "Dara");

        Console.WriteLine(students[2]);
    }
}
```

### Output

```text
Dara
```

# Summary

| Collection | Best Use |
|---|---|
| Array | Fixed-size data |
| List<T> | Dynamic data |
| Dictionary | Key-value lookup |
| Queue | FIFO processing |
| Stack | LIFO processing |
| HashSet | Unique values |
