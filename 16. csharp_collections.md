# C# Collections

## 1. Array

-   **Length**: Returns the number of elements
-   **Array.Sort()**: Sorts the array
-   **Array.Reverse()**: Reverses the array
-   **Array.IndexOf()**: Finds the index of an element
-   **Array.Clear()**: Clears values

``` csharp
int[] numbers = {30,10,20};
Console.WriteLine(numbers.Length);
Array.Sort(numbers);
foreach(var n in numbers)
    Console.WriteLine(n);
```

## 2. List`<T>`

| Method / Property | Description | Example |
|-------------------|-------------|----------|
| **Add()** | Adds one item to the end of the list. | `numbers.Add(10);` |
| **AddRange()** | Adds multiple items to the list. | `numbers.AddRange(new int[] {20, 30, 40});` |
| **Remove()** | Removes the first occurrence of a specified item. | `numbers.Remove(20);` |
| **RemoveAt()** | Removes an item at a specified index. | `numbers.RemoveAt(1);` |
| **Insert()** | Inserts an item at a specified index. | `numbers.Insert(0, 5);` |
| **Contains()** | Checks whether an item exists in the list. Returns `true` or `false`. | `numbers.Contains(30);` |
| **Sort()** | Sorts the list in ascending order. | `numbers.Sort();` |
| **Clear()** | Removes all items from the list. | `numbers.Clear();` |
| **Count** | Returns the number of items in the list. | `int total = numbers.Count;` |

## Example

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int>();

        // Add
        numbers.Add(10);
        numbers.Add(20);

        // AddRange
        numbers.AddRange(new int[] { 30, 40, 50 });

        // Insert
        numbers.Insert(1, 15);

        // Contains
        Console.WriteLine(numbers.Contains(30)); // True

        // Remove
        numbers.Remove(20);

        // RemoveAt
        numbers.RemoveAt(0);

        // Sort
        numbers.Sort();

        // Count
        Console.WriteLine("Total items: " + numbers.Count);

        // Display all elements
        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }

        // Clear
        numbers.Clear();

        Console.WriteLine("After Clear(): " + numbers.Count);
    }
}
```

## Output

```
True
Total items: 4
15
30
40
50
After Clear(): 0
```

## 3. Dictionary\<TKey,TValue\>

A `Dictionary<TKey, TValue>` stores key-value pairs and provides fast
lookups by key.

### Creating a Dictionary

``` csharp
using System.Collections.Generic;

Dictionary<int, string> students = new Dictionary<int, string>();
```

Or with initialization:

``` csharp
Dictionary<int, string> students = new Dictionary<int, string>
{
    { 1, "Alice" },
    { 2, "Bob" },
    { 3, "Charlie" }
};
```

### Adding Items

``` csharp
students.Add(4, "David");
```

Or:

``` csharp
students[5] = "Eve";
```

### Accessing Values

``` csharp
string name = students[1];
Console.WriteLine(name); // Alice
```

### Safe Lookup with TryGetValue

``` csharp
if (students.TryGetValue(2, out string studentName))
{
    Console.WriteLine(studentName);
}
```

### Checking for a Key

``` csharp
if (students.ContainsKey(3))
{
    Console.WriteLine("Key exists");
}
```

### Updating a Value

``` csharp
students[1] = "Anna";
```

### Removing an Item

``` csharp
students.Remove(2);
```

### Iterating Through a Dictionary

``` csharp
foreach (KeyValuePair<int, string> item in students)
{
    Console.WriteLine($"ID: {item.Key}, Name: {item.Value}");
}
```

Or:

``` csharp
foreach (var item in students)
{
    Console.WriteLine($"{item.Key}: {item.Value}");
}
```

### Common Operations

``` csharp
int count = students.Count;
students.Clear();
```

### Example

``` csharp
Dictionary<string, int> ages = new Dictionary<string, int>
{
    { "Alice", 25 },
    { "Bob", 30 }
};

ages["Charlie"] = 28;

foreach (var person in ages)
{
    Console.WriteLine($"{person.Key} is {person.Value} years old");
}
```

### Output

``` text
Alice is 25 years old
Bob is 30 years old
Charlie is 28 years old
```
