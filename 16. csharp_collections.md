# C# Collections

## 1. Array

-   **Length**: Returns the number of elements
-   **Array.Sort()**: Sorts the array
-   **Array.Reverse()**: Reverses the array
-   **Array.IndexOf()**: Finds the index of an element
-   **Array.Clear()**: Clears values

``` csharp
int[] numbers = {30,10,20};
Console.WriteLine(numbers.Length);
Array.Sort(numbers);
foreach(var n in numbers)
    Console.WriteLine(n);
```

## 2. List`<T>`



| Method / Property | Description | Example |
|-------------------|-------------|----------|
| **Add()** | Adds one item to the end of the list. | `numbers.Add(10);` |
| **AddRange()** | Adds multiple items to the list. | `numbers.AddRange(new int[] {20, 30, 40});` |
| **Remove()** | Removes the first occurrence of a specified item. | `numbers.Remove(20);` |
| **RemoveAt()** | Removes an item at a specified index. | `numbers.RemoveAt(1);` |
| **Insert()** | Inserts an item at a specified index. | `numbers.Insert(0, 5);` |
| **Contains()** | Checks whether an item exists in the list. Returns `true` or `false`. | `numbers.Contains(30);` |
| **Sort()** | Sorts the list in ascending order. | `numbers.Sort();` |
| **Clear()** | Removes all items from the list. | `numbers.Clear();` |
| **Count** | Returns the number of items in the list. | `int total = numbers.Count;` |

## Example

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int>();

        // Add
        numbers.Add(10);
        numbers.Add(20);

        // AddRange
        numbers.AddRange(new int[] { 30, 40, 50 });

        // Insert
        numbers.Insert(1, 15);

        // Contains
        Console.WriteLine(numbers.Contains(30)); // True

        // Remove
        numbers.Remove(20);

        // RemoveAt
        numbers.RemoveAt(0);

        // Sort
        numbers.Sort();

        // Count
        Console.WriteLine("Total items: " + numbers.Count);

        // Display all elements
        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }

        // Clear
        numbers.Clear();

        Console.WriteLine("After Clear(): " + numbers.Count);
    }
}
```

## Output

```
True
Total items: 4
15
30
40
50
After Clear(): 0
```
