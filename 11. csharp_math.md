# C# Math

The C# `Math` class provides many built-in methods for performing mathematical operations.

---

# Math.Max(x, y)

The `Math.Max(x, y)` method returns the largest value between two numbers.

## Syntax

```csharp
Math.Max(x, y);
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int result = Math.Max(5, 10);

        Console.WriteLine(result);
    }
}
```

## Output

```text
10
```

# Other Useful Math Methods

| Method | Description | Example |
|---|---|---|
| `Math.Pow(x, y)` | Returns x raised to the power y | `Math.Pow(2, 3)` → 8 |
| `Math.Ceiling(x)` | Rounds up | `Math.Ceiling(4.2)` → 5 |
| `Math.Floor(x)` | Rounds down | `Math.Floor(4.9)` → 4 |

---

# Full Example

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Max: " + Math.Max(5, 10));
        Console.WriteLine("Min: " + Math.Min(5, 10));
        Console.WriteLine("Sqrt: " + Math.Sqrt(64));
        Console.WriteLine("Abs: " + Math.Abs(-4.7));
        Console.WriteLine("Round: " + Math.Round(9.99));
    }
}
```

## Output

```text
Max: 10
Min: 5
Sqrt: 8
Abs: 4.7
Round: 10
```
