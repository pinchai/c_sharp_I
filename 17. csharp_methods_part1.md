# C# Methods

A **method** is a block of code that only runs when it is called.

You can pass data, known as **parameters**, into a method.

Methods are used to perform certain actions, and they are also known as **functions**.

## Why Use Methods?

Methods help you:

- Reuse code
- Make programs easier to read
- Reduce duplicate code
- Organize logic into smaller parts

---

# Create a Method

A method is defined with the name of the method, followed by parentheses `()`.

C# provides some built-in methods such as `Main()`, but you can also create your own methods.

## Syntax

```csharp
class Program
{
    static void MyMethod()
    {
        // code to be executed
    }
}
```

---

# Example Explained

```csharp
static void MyMethod()
```

### `MyMethod()`
- The name of the method.

### `static`
- Means the method belongs to the `Program` class instead of an object.

### `void`
- Means the method does not return any value.

> Note: In C#, method names usually start with an uppercase letter.

---

# Call a Method

To execute a method, write the method name followed by parentheses `()` and a semicolon `;`.

## Example

```csharp
using System;

class Program
{
    static void MyMethod()
    {
        Console.WriteLine("I just got executed!");
    }

    static void Main(string[] args)
    {
        MyMethod();
    }
}
```

## Output

```text
I just got executed!
```

---

# Calling a Method Multiple Times

A method can be called many times.

## Example

```csharp
using System;

class Program
{
    static void MyMethod()
    {
        Console.WriteLine("I just got executed!");
    }

    static void Main(string[] args)
    {
        MyMethod();
        MyMethod();
        MyMethod();
    }
}
```

## Output

```text
I just got executed!
I just got executed!
I just got executed!
```

---

# Method with Parameters

Parameters allow methods to receive data.

## Example

```csharp
using System;

class Program
{
    static void MyMethod(string name)
    {
        Console.WriteLine("Hello " + name);
    }

    static void Main(string[] args)
    {
        MyMethod("John");
        MyMethod("Sara");
    }
}
```

## Output

```text
Hello John
Hello Sara
```

---

# Method with Return Value

Use `return` to send a value back.

## Example

```csharp
using System;

class Program
{
    static int AddNumbers(int a, int b)
    {
        return a + b;
    }

    static void Main(string[] args)
    {
        int result = AddNumbers(5, 3);

        Console.WriteLine(result);
    }
}
```

## Output

```text
8
```

---

# Types of Methods

| Type | Description |
|---|---|
| `void` Method | Does not return a value |
| Return Method | Returns a value |
| Method with Parameters | Accepts input values |
| Static Method | Belongs to the class |

---

# Real-World Example

```csharp
using System;

class Program
{
    static double CalculateArea(double width, double height)
    {
        return width * height;
    }

    static void Main(string[] args)
    {
        double area = CalculateArea(5.5, 4.0);

        Console.WriteLine("Area = " + area);
    }
}
```

# Summary

- Methods are reusable blocks of code
- Methods run only when called
- Methods can receive parameters
- Methods can return values
- `void` means no return value
- `static` means the method belongs to the class
