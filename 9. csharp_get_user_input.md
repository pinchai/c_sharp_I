# C# Get User Input

`Console.ReadLine()` is used to get input from the keyboard in C#.

When the user types something and presses **Enter**, the value is returned as a **string**.

---

# Basic Example

```csharp
// Ask the user to enter a username
Console.WriteLine("Enter username:");

// Store user input in a variable
string userName = Console.ReadLine();

// Display the entered username
Console.WriteLine("Username is: " + userName);
```

---

# Output Example

```text
Enter username:
chai
Username is: chai
```

---

# User Input and Numbers

`Console.ReadLine()` always returns a `string`.

This will cause an error:

```csharp
Console.WriteLine("Enter your age:");
int age = Console.ReadLine(); // ERROR
```

Error:

```text
Cannot implicitly convert type 'string' to 'int'
```

---

# Convert String to Integer

Use `Convert.ToInt32()` to convert input into an integer.

```csharp
Console.WriteLine("Enter your age:");

// Convert string input to int
int age = Convert.ToInt32(Console.ReadLine());

Console.WriteLine("Your age is: " + age);
```

---

# Output Example

```text
Enter your age:
20
Your age is: 20
```

---

# Multiple User Inputs Example

```csharp
Console.Write("Enter your name: ");
string name = Console.ReadLine();

Console.Write("Enter your age: ");
int age = Convert.ToInt32(Console.ReadLine());

Console.WriteLine("\n--- User Information ---");
Console.WriteLine("Name: " + name);
Console.WriteLine("Age: " + age);
```

---

# Common Convert Methods

| Method | Description |
|---|---|
| `Convert.ToInt32()` | Convert to int |
| `Convert.ToDouble()` | Convert to double |
| `Convert.ToDecimal()` | Convert to decimal |
| `Convert.ToBoolean()` | Convert to bool |

---

# Example with Double

```csharp
Console.Write("Enter price: ");
double price = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Price is: " + price);
```

---

# Important Note

If the user enters the wrong data type, the program will throw an exception.

Example:

```text
Input string was not in a correct format.
```

Example problem:

```text
Enter age:
abc
```

This happens because `"abc"` cannot be converted to an integer.

---

# Safer Way Using TryParse

```csharp
Console.Write("Enter age: ");

string input = Console.ReadLine();

bool isNumber = int.TryParse(input, out int age);

if (isNumber)
{
    Console.WriteLine("Age is: " + age);
}
else
{
    Console.WriteLine("Invalid number input!");
}
```
