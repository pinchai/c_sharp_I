# C# Syntax

In the previous chapter, we created a C# file called `Program.cs`, and we used the following code to print `"Hello World"` to the screen.

## Example: Program.cs

```csharp
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }
}
```

## Output

```text
Hello World!
```

---

# Example Explained

## Line 1

```csharp
using System;
```

The `using System` statement allows us to use classes from the `System` namespace.

---

## Line 2

A blank line.

C# ignores white space, but using multiple lines makes code easier to read.

---

## Line 3

```csharp
namespace HelloWorld
```

A `namespace` is used to organize code. It acts as a container for classes and other namespaces.

---

## Line 4

```csharp
{
```

Curly braces `{ }` define the beginning and end of a block of code.

---

## Line 5

```csharp
class Program
```

A `class` is a container for data and methods.

Every line of executable C# code must be inside a class.

In this example, the class name is `Program`.

---

## Line 7

```csharp
static void Main(string[] args)
```

The `Main()` method is the entry point of a C# program.

Any code inside the `{ }` of `Main()` will run when the program starts.

---

## Line 9

```csharp
Console.WriteLine("Hello World!");
```

- `Console` is a class from the `System` namespace.
- `WriteLine()` is a method used to display text on the screen.

This line prints:

```text
Hello World!
```

---

# Important Notes

## Semicolon

Every C# statement ends with a semicolon `;`

Example:

```csharp
Console.WriteLine("Hello");
```

---

## Case-Sensitive Language

C# is case-sensitive.

These are different:

```csharp
MyClass
myclass
```

---

## File Name

Unlike Java, the filename does not need to match the class name.

However, using the same name is recommended for better organization.

Example:

```text
Program.cs
```

---

# Without `using System`

If you remove:

```csharp
using System;
```

You must write the full namespace path:

```csharp
System.Console.WriteLine("Hello World!");
```

---

# Simple Structure of a C# Program

```text
using System;

namespace MyApplication
{
    class Program
    {
        static void Main(string[] args)
        {
            // Code here
        }
    }
}
```

---

# Summary

| Part | Description |
|---|---|
| `using System` | Imports the System namespace |
| `namespace` | Organizes code |
| `class` | Container for methods and data |
| `Main()` | Program starting point |
| `Console.WriteLine()` | Prints text to the console |
| `;` | Ends a statement |
