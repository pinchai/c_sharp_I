# C# Operators

## Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

| Operator | Name | Example | Result |
|---|---|---|---|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Subtraction | `5 - 3` | `2` |
| `*` | Multiplication | `5 * 3` | `15` |
| `/` | Division | `10 / 2` | `5` |
| `%` | Modulus | `10 % 3` | `1` |
| `++` | Increment | `x++` | Increase by 1 |
| `--` | Decrement | `x--` | Decrease by 1 |

### Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int a = 10;
        int b = 3;

        Console.WriteLine(a + b);
        Console.WriteLine(a - b);
        Console.WriteLine(a * b);
        Console.WriteLine(a / b);
        Console.WriteLine(a % b);
    }
}
```

## Assignment Operators

| Operator | Example | Same As |
|---|---|---|
| `=` | `x = 5` | `x = 5` |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 3` | `x = x / 3` |
| `%=` | `x %= 3` | `x = x % 3` |

### Example

```csharp
int x = 10;

x += 5;
x -= 2;
x *= 2;

Console.WriteLine(x);
```

---

## C# Comparison Operators

| Operator | Name | Example |
|---|---|---|
| `==` | Equal to | `x == y` |
| `!=` | Not equal | `x != y` |
| `>` | Greater than | `x > y` |
| `<` | Less than | `x < y` |
| `>=` | Greater than or equal to | `x >= y` |
| `<=` | Less than or equal to | `x <= y` |

### Example

```csharp
int a = 10;
int b = 20;

Console.WriteLine(a == b);
Console.WriteLine(a != b);
Console.WriteLine(a < b);
Console.WriteLine(a > b);
```

---

## C# Logical Operators

| Operator | Name | Description |
|---|---|---|
| `&&` | Logical AND | True if both conditions are true |
| `||` | Logical OR | True if one condition is true |
| `!` | Logical NOT | Reverse the result |

### Example

```csharp
int age = 20;

Console.WriteLine(age > 18 && age < 30);
Console.WriteLine(age < 18 || age > 50);
Console.WriteLine(!(age > 18));
```
