# C# Conditions and If Statements

## C# Conditions

C# supports the following logical conditions:

- Equals to: `a == b`
- Not equal to: `a != b`
- Less than: `a < b`
- Greater than: `a > b`
- Less than or equal to: `a <= b`
- Greater than or equal to: `a >= b`

These conditions are often used in:

- `if`
- `else`
- `else if`

---

# The if Statement

Use `if` to specify a block of code to be executed if a condition is true.

## Syntax

```csharp
if (condition)
{
    // code to be executed
}
```

## Example

```csharp
int age = 20;

if (age >= 18)
{
    Console.WriteLine("You are an adult.");
}
```

### Output

```text
You are an adult.
```

---

# The else Statement

Use `else` to specify a block of code to be executed if the condition is false.

## Example

```csharp
int age = 15;

if (age >= 18)
{
    Console.WriteLine("You are an adult.");
}
else
{
    Console.WriteLine("You are under 18.");
}
```

### Output

```text
You are under 18.
```

---

# The else if Statement

Use `else if` to specify a new condition if the first condition is false.

## Example

```csharp
int score = 75;

if (score >= 90)
{
    Console.WriteLine("Grade A");
}
else if (score >= 80)
{
    Console.WriteLine("Grade B");
}
else if (score >= 70)
{
    Console.WriteLine("Grade C");
}
else
{
    Console.WriteLine("Failed");
}
```

### Output

```text
Grade C
```

---

# Short Hand If (Ternary Operator)

The ternary operator is a short-hand version of `if...else`.

## Syntax

```csharp
variable = (condition) ? expressionTrue : expressionFalse;
```

## Example

```csharp
int time = 20;

string result = (time < 18) ? "Good day." : "Good evening.";

Console.WriteLine(result);
```

### Output

```text
Good evening.
```

# Real-World Example

```csharp
Console.Write("Enter your password: ");
string password = Console.ReadLine();

if (password == "1234")
{
    Console.WriteLine("Login Success");
}
else
{
    Console.WriteLine("Wrong Password");
}
```

---

# Nested If Example

```csharp
int age = 25;
bool hasID = true;

if (age >= 18)
{
    if (hasID)
    {
        Console.WriteLine("Access Granted");
    }
    else
    {
        Console.WriteLine("ID Required");
    }
}
else
{
    Console.WriteLine("Access Denied");
}
```

# Logical Operators with If

| Operator | Meaning |
|---|---|
| `&&` | AND |
| `||` | OR |
| `!` | NOT |

## Example

```csharp
int age = 22;
bool hasTicket = true;

if (age >= 18 && hasTicket)
{
    Console.WriteLine("You can enter.");
}
```

# Summary

| Statement | Description |
|---|---|
| `if` | Executes code if condition is true |
| `else` | Executes code if condition is false |
| `else if` | Checks another condition |
| `?:` | Short-hand if else |
