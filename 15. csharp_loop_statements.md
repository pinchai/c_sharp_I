# C# Loop Statements

Loops are used to execute a block of code repeatedly.

---

# 1. `for` Loop

The `for` loop is used when you know how many times the loop should run.

## Syntax

```csharp
for(initialization; condition; increment/decrement)
{
    // code block
}
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        for(int i = 1; i <= 5; i++)
        {
            Console.WriteLine("Number: " + i);
        }
    }
}
```

---

# 2. `while` Loop

The `while` loop repeats as long as the condition is `true`.

## Syntax

```csharp
while(condition)
{
    // code block
}
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int i = 1;

        while(i <= 5)
        {
            Console.WriteLine("Value: " + i);
            i++;
        }
    }
}
```

---

# 3. `do while` Loop

The `do while` loop executes the code block at least once before checking the condition.

## Syntax

```csharp
do
{
    // code block
}
while(condition);
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        int i = 1;

        do
        {
            Console.WriteLine("Count: " + i);
            i++;
        }
        while(i <= 5);
    }
}
```

---

# 4. `foreach` Loop

The `foreach` loop is used to loop through arrays or collections.

## Syntax

```csharp
foreach(dataType variable in collection)
{
    // code block
}
```

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        string[] fruits = {"Apple", "Banana", "Orange"};

        foreach(string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}
```

---

# 5. `break` Keyword

The `break` keyword stops the loop immediately.

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        for(int i = 1; i <= 10; i++)
        {
            if(i == 5)
            {
                break;
            }

            Console.WriteLine(i);
        }
    }
}
```

---

# 6. `continue` Keyword

The `continue` keyword skips the current iteration and continues with the next iteration.

## Example

```csharp
using System;

class Program
{
    static void Main()
    {
        for(int i = 1; i <= 5; i++)
        {
            if(i == 3)
            {
                continue;
            }

            Console.WriteLine(i);
        }
    }
}
```
